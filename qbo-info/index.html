AUSOMA Student Finance Reconciliation Tool

Internal application — American University School of Medicine of Aruba

This application is an internal financial reconciliation tool operated by American University School of Medicine of Aruba (AUSOMA). It performs a daily three-way match across AUSOMA's enrollment records, QuickBooks Online invoices and payments, and Populi student-information-system invoices and payments, producing exception reports for AUSOMA's bursar office.

Scope & access

The tool is operated by authorized AUSOMA staff only. It is not a multi-tenant product and is not offered to any party outside AUSOMA. There is no public registration; access is provisioned individually by AUSOMA's Office of the Registrar and Office of Academic Affairs.

What it connects to

The tool reads data via authenticated API connections from:

  • AUSOMA's QuickBooks Online company file (read-only OAuth 2.0 scope)
  • AUSOMA's Populi tenant (read-only API key)
  • AUSOMA's Stripe account (read-only restricted key)

It does not write to any of these systems. Data is processed in memory or on AUSOMA-controlled infrastructure and is never transmitted to any party outside AUSOMA or its authorized service providers.

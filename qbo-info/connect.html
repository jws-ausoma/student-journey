Connect to QuickBooks Online

For authorized AUSOMA staff only

The AUSOMA Student Finance Reconciliation Tool connects to AUSOMA's QuickBooks Online company file via OAuth 2.0 with read-only scope (com.intuit.quickbooks.accounting).

Initial connection

The connection is established once by an authorized AUSOMA staff member with administrator access to AUSOMA's QuickBooks Online account. The OAuth bootstrap is performed on AUSOMA-controlled infrastructure during the tool's initial setup.

Reconnection

QuickBooks Online refresh tokens are rotated regularly per Intuit's OAuth 2.0 specification. If a refresh fails or a token is revoked, an authorized AUSOMA staff member must re-run the OAuth bootstrap to establish a new connection. The reconnection flow uses the same OAuth 2.0 authorization code grant as the initial connection.

To request a connection

Contact AUSOMA's Office of the Registrar: jawad.shaikh@ausoma.org. External requests cannot be processed — this is an internal AUSOMA tool.
